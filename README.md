# Column Extractor

# Run `csvextractor`
## Options: 
* -h / ? / -help / --help
* -c / --column
* -f / --file
* -u / --unknown

# Example
## Unknown Column
## ./bin/com.riversidecode.extractor csvextractor -f <file location> -u

## Known Column
## ./bin/com.riversidecode.extractor csvextractor -f <file location> -c 10
